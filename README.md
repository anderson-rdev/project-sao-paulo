## Funcionalidades

- **Navbar responsiva** com scroll suave e efeito sticky
- **Carousel Bootstrap** com 6 slides e animações CSS (`fadeUp`)
- **Cards de conquistas** com hover interativo e reveal por scroll (IntersectionObserver)
- **Timeline** histórica dos momentos mais marcantes do clube
- **Design temático** nas cores oficiais do São Paulo FC (`#e30613`, `#0a0a0a`, `#f5f5f0`)
- **Totalmente responsivo** — mobile, tablet e desktop
- **Zero dependências de back-end** — arquivo HTML único

## Como usar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/saopaulo-fc.git

# Abra no navegador
cd saopaulo-fc
open index.html
```

Ou simplesmente faça o download do `index.html` e abra direto no navegador — sem necessidade de servidor.

## Deploy

Compatível com qualquer hospedagem estática:

- **GitHub Pages** — Settings → Pages → Deploy from branch
- **Vercel** — importe o repositório e faça deploy com um clique
- **Netlify** — arraste o arquivo `index.html` para o painel
